package com.haxepunk.masks

Circle

Uses circular area to determine collision.

Grid

Uses a hash grid to determine collision, faster than using hundreds of Entities for tiled levels, etc.

Hitbox

Uses parent's hitbox to determine collision. This class is used internally by HaxePunk, you don't need to use this class because this is the default behaviour of Entities without a Mask object.

Masklist

A Mask that can contain multiple Masks of one or various types.

Pixelmask

A bitmap mask used for pixel-perfect collision.

Polygon

Uses polygonal structure to check for collisions.

SlopedGrid

Uses a hash grid to determine collision, faster than using hundreds of Entities for tiled levels, etc.
