Engine

Main game Sprite class, added to the Flash Stage. Manages the game loop.

Entity

Main game Entity class updated by Scene.

Graphic

Base class for graphics type. Do not use this directly, instead use the classes in com.haxepunk.graphics.

HXP

Static catch-all class used to access global properties and functions.

ImageType

Abstract representing either a String, a TileAtlas, a BitmapData or a AtlasRegion.

Mask

Base class for Entity collision masks. Do not use this directly, instead use the classes in com.haxepunk.masks.

Position

Represents a position on a two dimensional coordinate system.

RenderMode

The render mode used by HaxePunk. Normal mode for flash&html5 is buffer, others are hardware. To change the mode used see the Engine.new constructor, but beware that things may not work if you do that.

Scene

Updated by Engine, main game container that holds all currently active Entities. Useful for organization, eg. "Menu", "Level1", etc.

Screen

Container for the main screen buffer. Can be used to transform the screen. To be used through HXP.screen.

SolidType

Abstract representing either a String or a Array<String>.

TileType

Abstract representing either a String, a TileAtlas or a BitmapData.

Tween

Base class for tweening helpers. Do not use this directly, instead use the classes in com.haxepunk.tweens.

TweenType

The type of the tween.

Tweener

Abstract class used to add the ability to add tweens.
