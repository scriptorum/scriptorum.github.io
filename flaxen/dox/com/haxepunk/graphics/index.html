package com.haxepunk.graphics

atlas
Animation

Template used by Spritemap to define animations. Don't create these yourself, instead you can fetch them with Spritemap.add.

Backdrop

A background texture that can be repeated horizontally and vertically when drawn. Really useful for parallax backgrounds, textures, etc.

Canvas

A multi-purpose drawing canvas, can be sized beyond the normal Flash BitmapData limits. Works only on flash and html5 targets.

Emitter

Particle emitter used for emitting and rendering particle sprites. Good rendering performance with large amounts of particles.

Graphiclist

A Graphic that can contain multiple Graphics of one or various types. Useful for drawing sprites with multiple different parts, etc.

Image

Performance-optimized non-animated image. Can be drawn to the screen with transformations.

ParticleType

Template used to define a particle type used by the Emitter class. Instead of creating this object yourself, fetch one with Emitter.add function.

Spritemap

Performance-optimized animated Image. Can have multiple animations, which draw frames from the provided source image to the screen.

StyleType

Abstract representing either a TextFormat or a TextOptions.

Text

Used for drawing text using embedded fonts.

TextOptions

Text option including the font, size, color...

Tilemap

A canvas to which Tiles can be drawn for fast multiple tile rendering.
