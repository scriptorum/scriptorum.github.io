class flaxen.system.DeathBoxSystem extends FlaxenSystem

Available on all platforms

A death box is an area on the screen for automatically removing entities that either stray out of or into specified areas.

An entity must possess Position and a DeathBox to be susceptible to this system.

To use this system, it must be added to Ash: flaxen.addSystem(new DeathBoxSystem()). It should probably be added after the flaxen.system.MovementSystem.
