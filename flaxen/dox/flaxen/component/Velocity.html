class flaxen.component.Velocity

Available on all platforms

Velocity indicates the distance moved in one second, applied to a flaxen.component.Position component.

This component is processed by the flaxen.system.MovementSystem.

function lessThan(i:Float):Bool

This convenience method checks if both axes are within a certain distance to 0. That is, if x is 5 and y is -5, lessThan(6) will return true, but lessThan(5) will return false.

  • TODO: Users of this method may be more interested in checking the magnitude of x/y as a whole, rather than each individual axis. Worth considering.
