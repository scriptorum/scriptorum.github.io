class flaxen.component.Image

Available on all platforms

Represents a path (URL) to an image file and optional cropping rectangle (clip).

Width and Height will be set by Flaxen after it is processed by the rendering system.

Same thing for clipWidth/Height. If an ImageGrid is applied, these will represent the tile dimensions. Otherwise, it will be equal to the clip.width/height.
