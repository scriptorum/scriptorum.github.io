class flaxen.component.Image

Available on all platforms

Represents a path (URL) to an image file and optional cropping rectangle (clip).

Regarding clipWidth/clipHeight: This is set by RenderingSystem after image is processed and is READ-ONLY. If an ImageGrid is shared by the entity, this represents the tile dimensions. If a clip rectangle is supplied, this represents the clipped dimensions. Otherwise, this is the same as width.

  • QUESTION: Is clipWidth/Height really necessary?
  • QUESTION: In fact should I even store width/height?
