enum flaxen.component.OnCompleteAnimation

import flaxen.component.Animation;

Available on all platforms

When an animation "completes," what action should we take? This enum is a custom extension of flaxen.common.OnComplete. An animation completes when it finishes its sequence (if LoopType is None) or when the looping is interrupted by setting stop to true.

  • Note: Pausing the animation with paused=true will not set the complete flag. Setting OnComplete to Paused

Constructors

DestroyEntity

Destroy the entity holding this component

DestroyComponent

Destroy the component, removing it from the entity

Clear

The animation disappears (default).

Last

The animation freezes on the last frame

First

The animation freezes on the first frame

Current

The animation remains on its last frame where it was manually stopped
