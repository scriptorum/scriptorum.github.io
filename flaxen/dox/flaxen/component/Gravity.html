class flaxen.component.Gravity

Available on all platforms

Gravity indicates the change of flaxen.component.Velocity over one second towards a particular axis end (usually down).

This is a low-rent Gravity implementation. It must be added to each entity that experiences gravity, so it's not very useful as a broad spectrum force. You must implement your own system to handle landing on floors/surfaces, or going off-screen.

This component is processed by the flaxen.system.GravitySystem.
